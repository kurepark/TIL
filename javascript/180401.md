### 값
값의 특성 : 피연산자, 식의 결과, 변수에 할당이 가능하다
값의 종류
- 숫자 number
- 문자 string
- 참, 거짓 boolean
- 배열 object
- 객체 object
- 함수 function
- 기타 null(object), undefined(undefined), NaN(number), Infinity(number)...

### 함수
함수의 호출 : 함수명 ();
함수의 선언      
function 함수명(){     
    //함수본문(식, 값, 문)      
}       

함수는 선언만으로는 아무일도 일어나지 않는다. 선언과 호출이 함께.

```
sum(5, 2);//함수의 호출

function sum(x, y){
    return x + y;
}//함수의 선언

var abc = sum();//함수는 값이기 때문에 변수에 할당이 가능하다
abc(); //호출도 가능
```
