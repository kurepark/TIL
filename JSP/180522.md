### 인프런JSP 3강
-----

##### JSP특징


   - 동적 웹 어플리케이션 컴포넌트
   - .jsp확장자
   - 클라이언트의 요청에 동적으로 작동하고 응답은 html을 이용
   - jsp는 서블릿으로 변환되어 실행
   - MVC(Model, View, Controller)패턴에서 View로 이용됨

   클라이언트 - request - > Controller(servlet) -> Model -> Controller -> View(JSP) - response -> 클라이언트
